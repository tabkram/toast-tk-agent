--- 
Done by Nathan Cohen
The 11/08/2016
---

Toast TK agent have been installed in 64 bits.

The installation of the agent created some files in the directory "%user\{username}\.toast\" and the agent launcher in the file you choosed.

Every file installed will be erase if you remove the program in the Add/Remove Program Control Panel.

For more information, go to http://toast-tk.io/.
